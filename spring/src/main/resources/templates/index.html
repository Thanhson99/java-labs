What This Backend Teaches

Security

JWT access tokens, refresh rotation, logout, logout-all, reuse detection.

Data

Primary user database, analytics database, Flyway migrations, transaction rollback.

Messaging

Registration events published to Kafka and RabbitMQ with shared consumer processing.

Testing

Controller tests, service tests, Testcontainers for PostgreSQL integration.

Useful Endpoints

  • GET/hello?name=Spring
  • POST/api/auth/token
  • POST/api/auth/refresh
  • POST/api/auth/logout
  • POST/api/auth/logout-all
  • POST/api/users/register
  • GET/api/system/overview
  • GET/h2-console

Suggested Learning Path

  1. Open the root page and inspect the template, CSS, and script.
  2. Read `RegistrationController` and `RegistrationService`.
  3. Request a token, then call protected endpoints.
  4. Read the auth and messaging tests after changing one behavior.
