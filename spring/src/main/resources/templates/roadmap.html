Milestones

Each item is tied to code already present in this repo.

Java variables, parameters, and methods

Start with syntax and signatures. Use the cheat sheet and the basic module entry points.

Control flow and loops

Learn if, switch, for, and for-each.

Classes, records, encapsulation, and OOP

Study domain classes and small immutable DTOs.

Collections, streams, and analytics logic

Read how data is grouped, sorted, and summarized in memory.

JDBC and database access

Connect Java code to SQL, then compare it with Spring Data and JdbcTemplate.

Spring MVC controllers and templates

Follow request flow from page to controller to service.

JWT auth, refresh tokens, and sessions

Understand access token issuance, rotation, revocation, and reuse detection.

Transactions, rollback, and Flyway

Study why schema migration and transactional writes belong together.

Outbox pattern and durable event publishing

Read the write path, outbox store, dispatcher, and dead-letter handling.

Kafka and RabbitMQ producers and consumers

Compare transport adapters and how the same event reaches different systems.

Microservice boundaries and notification service

Understand the split between the main app and the extracted notification service.

Observability and admin operations

Use metrics, health checks, admin console, and runtime diagnostics like an operator.
